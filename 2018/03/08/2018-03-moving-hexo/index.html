迁移hexo

2018-03-08
  • 博客维护

重装了工作站，使用2×2硬盘电源切换器完成了win7/ubuntu17双启动
然后立即面临hexo for win —> hexo for ubuntu的问题，是这么解决的：

打包hexo文件夹，其中包含myblog目录

在unbuntu上安装npm
在/home/username 下解包hexo.zip
在/home/username/hexo/myblog下执行 npm install
然后 hexo d -g
然后回答问题就可以了。
